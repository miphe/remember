Prototype #1
====

In this first prototype users will be able to:

- identify themselves
- write new entries
- edit their own entries
- search

On the topic of visibility / access control:

- must be logged in to read / write entries
- when logged in, you can see everyone else's entries and they can see all of yours.

On the topic of writing / editing entries:

- autosuggest for the `title` field based on previous entries
- basic markdown syntax
- no builtin tagging support
